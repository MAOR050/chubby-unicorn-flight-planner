Chubby Unicorn - Standalone PWA Version

This folder contains the complete Progressive Web App that can be:
1. Served by any web server
2. Installed as a PWA in modern browsers
3. Run offline after installation

To use:
1. Open a terminal in this directory
2. Run: python3 -m http.server 8080
3. Open browser to: http://localhost:8080
4. Install as PWA using browser's install prompt
